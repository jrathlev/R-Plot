Schnelleinstieg: X-Y-Diagramm erstellen

  1. Neues Arbeitsblatt anlegen

    • Größe auswählen
    • Ränder einstellen (nur für direkten Ausdruck erforderlich)
    • Hintergrundfarbe auswählen

  2. Neue Kurve hinzufügen

    • Es wird automatisch ein neues Diagramm so einfügt, dass es das Arbeitsblatt vollständig ausfüllt. Änderungen sind später jederzeit möglich.
    • Es werden automatisch eine X- und eine Y-Achse mit den vorgegebenen Einstellungen angelegt. Die Bereichsanpassung an die Datenkurve erfolgt ebenfalls automatisch.
    • In dem Dialog für die Datenkurve werden die Messdaten nach Klick auf die Schaltfläche Daten bearbeiten entweder von Hand eingegeben oder aus einer CSV-Datei importiert. Anschließend können noch die Anzeigeeigenschaften der Datenkurve individuell angepasst werden.
      Nach Klick auf die Schaltfläche OK wird das neue Diagramm angezeigt.

Schnelleinstieg: Funktion im X-Y-Diagramm darstellen

  1. Neues Arbeitsblatt anlegen

    • Größe auswählen
    • Ränder einstellen (nur für direkten Ausdruck erforderlich)
    • Hintergrundfarbe auswählen

  2. Neue Funktion hinzufügen

    • Es wird automatisch ein neues Diagramm so einfügt, dass es das Arbeitsblatt vollständig ausfüllt. Änderungen sind später jederzeit möglich.
    • Es wird automatisch eine neue X-Achse mit den vorgegebenen Einstellungen angelegt. Der Bereich wird entsprechend der gewünschten Darstellung gewählt.
    • Die Y-Achse wird automatisch mit den vorgegebenen Einstellungen angelegt. Die Bereichsanpassung an die Funktion erfolgt ebenfalls automatisch.
    • In dem Dialog für die Funktion wird die Formel der darzustellenden Funktion eingegeben. Anschließend können noch die Anzeigeeigenschaften der Funktionskurve individuell angepasst werden.
      Nach Klick auf die Schaltfläche OK wird das neue Diagramm angezeigt.
